рЅаріарѕѕрѕЏрЅйріЋ рѕІрІГ рІѕрІ░ рѕўрЅХ рІерѕЮрїарїЅ рІерЇЋрѕ«рїЇрѕФрѕЮріЋрїЇ рЅІріЋрЅІрІјрЅй ріарѕЅрЇб рїЇріЋ Programming рѕЮріЋ рѕЏрѕѕрЅх ріљрІЇ?
рІерѕ░рІЇ рѕЇрїєрЅй рЅарЅІріЋрЅІ ріЦріЋрІ░рѕЮріЋрїЇрЅБрЅБрІЇ ріері«рѕЮрЇњрІЇрЅ░рѕГ рїІрѕГ рѕѕрѕўрїЇрЅБрЅБрЅх (рІерЅ░рѕѕрІФрІЕ рЅхрІЋрІЏрІЮ рѕѕрѕўрѕхрїарЅх) рѕѕрЅЦрІЎ ріарѕўрЅ│рЅх рѕѕрЅ░рѕўрѕФрѕЏрѕфрІјрЅй рІерѕФрѕх рѕЮрЅ│рЅх ріљрЅарѕГ.. рѕЮ/рЅ▒рѕЮ рі«рѕЮрЇњрІЇрЅ░рѕ«рЅй рѕЮрІФрІЇрЅЂрЅх ­ЪЉЅ 0101001 (binary numbers) ріљрІЇ ріЦріЏ рІ░рїЇрѕъ рІГрѕёріЋріЋ рѕўрѕерІ│рЅх ріаріЋрЅйрѕЇрѕЮ ­ЪциРђЇРЎѓ№ИЈ!! рѕхрѕѕрІџрѕЁ рѕЂрѕѕрЅ│рЅйріЋріЋ рІерѕЮрІФрїЇрЅБрЅА high level programming рЅІріЋрЅІрІјрЅй рЅ░рЇѕрїарѕЕрЇб рЅарѕўрѕєріЉрѕЮ ріЦріћ рѕѕрѕЮрѕ│рѕї cout<<"hi" рЅЦрІг рЅЦрЇЁрЇЇ рі«рѕЮрЇЊрІГрѕѕрѕЕ (compiler) рѕѕрі«рѕЮрЇњрІЕрЅ░рѕг рІўрІ░ 101010 рЅђрІГрѕ« рІГрїЇрѕерІІрѕЇ рЇб ­ЪцЮ

рІе C++ рЅБрѕЁрѕф ріЦріЊ рЅ░рїЇрЅБрѕФрЅх

1. C++ is general purpose programming language.

C++ рѕѕрїарЅЁрѕІрѕІ рїЦрЅЁрѕЮ рІерѕџрІЇрѕЇ рІерЇЋрѕ«рїЇрѕФрѕЮріЋрїЇ рЅІріЋрЅІ ріљрІЇрЇц рѕЏрѕѕрЅхрѕЮ ріерЅфрІ▓рІ« рїїрѕЮ ріЦрѕхріе рі«рѕЮрЇњрІЇрЅ░рѕГ ріЦріЊ рІерѕърЅБрІГрѕЇ ріарЇЋрѕірігрѕйріЋрЇБ ріерѕЎрІџрЅЃ рѕЏрЅђріљрЅБрЅарѕфрІФ ріарЇЊрЅй ріЦрѕхріе рІерЅ░рІѕрѕ│рѕ░рЅА рІеAI рѕХрЇЇрЅхрІїрѕ«рЅй рѕЂрѕЅріЋрѕЮ рѕўрѕхрѕФрЅх рІФрѕхрЅйрѕІрѕЇрЇб

2. C++ is Object Oriented

рІГрѕЁ рЅа programming рІЇрѕхрїЦ ріЦрїЁрїЇ рЅарїБрѕЮ рІѕрѕ│ріЮ Concept ріљрІЇ (рІерѕЮрѕгріЋ ріљрІЇ) рІѕрІ░рЇірЅх рЅарѕ░рЇірІЇ ріЦріЊрІФрѕѕріЋрЇб рїЇріЋ рЅаріарїГрѕЕ рѕѕрѕЏрЅЦрѕФрѕФрЅх рІФрѕЁрѕЇ (рѕѕрѕЮрѕ│рѕї ріљрІЇ) OOP (Object Oriented programming) рІЇрѕхрїЦ ріљрїѕрѕ«рЅйріЋ рѕЇріГ ріЦріЋрІ░ ріЦрІЇріљрЅ░ріЏ Object ріљрІЇ рІерѕЮріЊрѕхрѕўрѕхрѕѕрІЇ рЇб ріЦріЋрЅарѕЇріЊ рЅа C++ рІЇрѕхрїЦ ріаріЋрІх рѕхрѕѕ 20 рѕўріфріќрЅй рІерЅђрѕѕрѕЮ ріарІГріљрЅхріЊ рЇЇрїЦріљрЅх рІерѕЮріЊрїѕрѕГ рі«рІх рѕѕрѕўрЇЃрЇЇ рѕѕрѕЂрѕЅрѕЮ рѕўріфріќрЅй рІерЅ░рѕѕрІФрІг рі«рІх ріаріЋрЇЁрЇЇрѕЮ! рЅарЅЃ ріаріЋрІх "car" рѕЮрЅБрѕЇ class рЅарѕўрЇЇрїарѕГ color рІерѕЮрЅБрѕЇ variable , speed () рІерѕЮрЅБрѕЇ function рЅарІЇрѕхрїА рЅарѕўрїерѕўрѕГ ef class Car { public : public: void speed(); string color; } рІГрѕёріЋріЉ class рѕѕрЅЦрІЎ object(рІФрІЇ рЅарІџрѕЁ рі«рІх рІеріЏ object рѕўріфріЊрІЇ ріљрІЇ) рѕўрїарЅђрѕЮ рЅйрѕІрѕѕрІЇрЇб Car car1; Car car2; Car car20; ...... ріЦрІФрѕЇріЕ рѕЏрѕѕрЅх ріљрІЇ рІѕрІ░рЇірЅх рІГрїѕрЅБрЅйріІрѕЇ рїЇріЋ рЅарїБрѕЮ рІѕрѕ│ріЮ ріљрІЇ ­Ъўі
